h1. mlgenotype

p. This python package can be used to train machine learning models to genotype
structural variants using aligned short read data. It was written by Nancy Fisher Hansen,
a staff scientist in the Cancer Genetics and Comparative Genomics Branch (CGCG) of NHGRI,
starting with code written by Gracelyn Hill and
Jennifer C Lin.  Nancy can be reached at nhansen@mail.nih.gov.

h2. Software dependencies

* pandas
* sklearn

h3. Clone the repository

bc. git clone git://github.com/nhansen/mlgenotype
